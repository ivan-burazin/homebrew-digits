# homebrew-digits

Homebrew tap for the [Digits](https://github.com/ivan-burazin/digits) CLI.

```bash
brew tap ivan-burazin/digits
brew install digits
```

Formulas in this repo are maintained automatically by GoReleaser on
every tagged release of the digits repo. Don't edit `Formula/digits.rb`
by hand — it'll be overwritten on the next release.
